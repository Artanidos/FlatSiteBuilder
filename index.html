FlatSiteBuilder

The FlatSiteBuilder is a very easy to use content management system and content creation tool running as a desktop application on Linux.

It is also planned to release versions for Windows and Mac OS.

Fatest CMS

Because of the fact the FlatSiteBuilder only creates flat HTML files it is therefore the fatest CMS available.

Easiest CMS

Because of the fact that FlatSiteBuilder runs on the desktop it is therefore one of the easiest to use CMS avaiable.

Safest CMS

Because of the fact that we use github as version control for the content, FlatSiteBuilder is therefore the safest CMS available.

Cheapest CMS

Because of the fact that hosting on github pages is free, FlatSiteBuilder is therefore the cheapest CMS available.
