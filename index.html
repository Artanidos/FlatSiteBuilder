Welcome to FlatSiteBuilder

Did you knew that you are able to host your project website on Github-Pages for free?

The FlatSiteBuilder is a very easy to use content management system (CMS) and content creation tool running as a desktop application on Linux. You are able to build web content which you can host on github pages for free or any other web space provider. Because all of the pages are beeing build on the desktop it is the fastest content management system available. No code is executed on the web server.

FlatSiteBuilder

The FlatSiteBuilder is a very easy to use content management system and content creation tool running as a desktop application on Linux.

It is also planned to release versions for Windows and Mac OS.

Fastest CMS

Because of the fact the FlatSiteBuilder only creates flat HTML files it is therefore the fastest CMS available.

Easiest CMS

Because of the fact that FlatSiteBuilder runs on the desktop it is therefore one of the easiest to use CMS avaiable.

Safest CMS

Because of the fact that we use github as version control for the content, FlatSiteBuilder is therefore the safest CMS available.

Cheapest CMS

Because of the fact that hosting on github pages is free, FlatSiteBuilder is therefore the cheapest CMS available.
