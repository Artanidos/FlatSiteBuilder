Introduction

This version of FlatSiteBuilder is a preview to show possibilities.
It is not meant to be a ready made product.
There are still some features not implemented yet.

If you for example want to change site properties like the menus just edit the Site.xml associated with the website and reload it.
Also if you want to change a theme or want to override theme assets or includes you have to do it by hand.
Have a look at ~/FlatSiteBuilder/
